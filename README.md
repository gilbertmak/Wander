# FIRE Planner And Statement Ingestion

This workspace is the implementation target for a FIRE planner that combines:

- FIRE planning projections based on the `RemarkRemedy/fireplanner` reference product.
- Bank statement ingestion based on the `benjamin-awd/StatementSensei` reference product.
- Singapore credit-card miles calculation, refund handling, MCC tagging, and card optimization.

Start with [docs/IMPLEMENTATION_HANDOFF.md](docs/IMPLEMENTATION_HANDOFF.md). It contains the epics, architecture decisions, database plan, UI direction, traceability matrix, implementation sequence, and validation checklist required for another model or coding agent to implement the product in small reviewable slices.
